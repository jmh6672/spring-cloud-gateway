# API 게이트웨이

## 목적
- API 호출시 인증 및 인가 체크
- API tracing
- Ingress service

## 패키지 그룹
`common-module`의 AOP 를 피하기 위해서. 타 프로젝트와 다르게 패키지 그룹에서 `gateway` 경로가 추가 되어있다.
